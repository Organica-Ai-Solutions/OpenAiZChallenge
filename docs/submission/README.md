## üîß API Documentation

### Core Endpoints

```bash

POST /analyze

  "lat": -9.8000,
  "lon": -84.2000,
  "data_sources": ["satellite", "lidar", "historical"],
  "confidence_threshold": 0.7
}

# Get analysis results
GET /research/sites?min_confidence=0.8&max_sites=50

POST /lidar/data/latest
```
